## Welcome to Mindlap Social Scheduler

Mindlap Social Scheduler is a powerful backend API system designed to automate and optimize your social media presence across multiple platforms. Built for Mindlap.ai's voice AI platform, it combines intelligent scheduling, AI-powered content generation, and comprehensive analytics.

## Technology Stack

- **Backend**: Node.js + Express.js
- **Database**: SQLite (dev) / PostgreSQL (prod)
- **AI**: OpenAI GPT-4o-mini
- **Scheduling**: node-cron
- **Authentication**: JWT + bcrypt
- **Social APIs**: Twitter V2, LinkedIn, Instagram, YouTube
